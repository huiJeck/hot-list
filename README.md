
## 项目设置

```sh
npm install
```

### 在开发环境中启动和热更新

```sh
npm run dev
```

### 编译用于生产环境

```sh
npm run build
```

### 说明
这是个人为了方便查看各平台实时热榜新闻而开发，支持pc端和移动端、指在达到10分钟了解所有热点时事资讯，过滤掉无效资讯。
